# testClicoh

## API with django rest framework.

### Installation

Use the requirements file to install all dependencies.

```bash
pip install -r requirements.txt
```

### API Usage
#### Authentication
##### Generate a token /api/token/


| Description   | Creates a new token        |
| Method        | POST                       |
| Body                                       |
|         |Key       | Value                 |
|         |username  | <username>            | 
|         |password  | <password>            |
